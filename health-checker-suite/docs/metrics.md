# Metrics

The metrics module contains standardized algorithms:

- `HealthScoreCalculator`: Deducts points based on severity weights.
- `ComplexityCalculator`: Aggregates cyclomatic and cognitive complexity.
- `StatisticsCollector`: Counts files, total lines, blank lines, and comments.
- `QualityMetricsCalculator`: Calculates defect density.
- `ExecutionTimingMetrics`: Measures performance.

# Backend Health Checker

A professional, multi-threaded, AST-based health checker for large Python projects.

## Features
- Syntax & Style Checking
- Advanced Import Analysis (Cycles, Unused, Wildcards)
- Configurable Architecture Layer Verification
- Dead Code & Duplicate Detection
- Security Scanning
- Standalone HTML Dashboard & Rich CLI
- Health Score (0-100)

## Installation
```bash
pip install -e .
```
